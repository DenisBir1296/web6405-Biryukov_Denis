# web6405-Biryukov_Denis
LAB3
Лабораторная работа содержит ряд функций для которых необходимо написать реализацию. В числе которых функции для создания генераторов, вычисления рядов и проверки чисел на заданные условия. Наибольшее затруднее вызвала функция deepEqual из-за большого колличества проверок и наличия вложенности в обрабатываемых объектах. Менее затруднительными оказались isInteger и isBinary. Для запуска тестов функций использовалась среда Node js. Необходимые зависимости были установлены с помощью пакетного менеджера npm.
